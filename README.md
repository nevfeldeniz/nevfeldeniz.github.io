# Nevfel Deniz Sürmeli — Kişisel Portföy

Modern, karanlık temalı kişisel portföy web sitesi.

## Teknolojiler

- [Next.js 14](https://nextjs.org/) (App Router)
- React 18 + TypeScript
- Tailwind CSS
- Framer Motion (animasyonlar)
- lucide-react (ikonlar)

## Geliştirme

```bash
npm install
npm run dev
```

Tarayıcıda [http://localhost:3000](http://localhost:3000) adresini açın.

## İçerik düzenleme

Tüm metinler tek dosyada yönetilir: `src/data/portfolio.ts`

## Yayınlama

Bu proje [Vercel](https://vercel.com) üzerinde yayınlanmak üzere hazırlanmıştır.
GitHub deposu Vercel'e bağlandığında her `git push` otomatik olarak siteyi günceller.

## Bölümler

- Hakkımda
- Kariyer ve Eğitim Yolculuğu (zaman çizelgesi)
- Yetenekler
